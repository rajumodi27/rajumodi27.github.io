## Speech Recognition Implementation

The application uses `speech_recognition` to capture voice input through the microphone. Here's how the core speech recognition works:

```python
def capture_voice():
    recognizer = speech_recognition.Recognizer()
    with speech_recognition.Microphone() as source:
        print("Listening...")
        audio = recognizer.listen(source)
        return recognizer.recognize_google(audio)
```

## Translation Engine

The translation engine supports multiple languages through the `translate` library:

```python
def translate_text(text, target_lang):
    translator = Translator(to_lang=target_lang)
    return translator.translate(text)
```

## Text-to-Speech Output

Voice output is handled by gTTS for natural-sounding translations:

```python
def speak_translation(text, language):
    tts = gTTS(text=text, lang=language)
    tts.save("translation.mp3")
    playsound.playsound("translation.mp3")
```
