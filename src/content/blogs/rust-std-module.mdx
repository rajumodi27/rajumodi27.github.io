## 1. Introduction to `std`
The Rust Standard Library (`std`) provides essential functionality such as file handling, networking, collections, and concurrency. It is **built-in and included by default**.

### How to Use `std`?
You can use Rustâ€™s `use` keyword to bring specific modules into scope:

```tsx

</CodeBlock>
```


## **2. File Handling (`std::fs`)**
The `fs` module provides functionality for **reading and writing files**.

### **Reading a File**
```tsx
<CodeBlock language="rust">
{`use std::fs;

    let content = fs::read_to_string("example.txt").expect("Failed to read file");

</CodeBlock>
```

### **Writing to a File**
```tsx

}`}
</CodeBlock>
```

## **3. Input/Output (`std::io`)**
The `io` module provides functions for **handling user input and printing output**.

### **Reading from Standard Input (stdin)**
```tsx
<CodeBlock language="rust">
{`use std::io;

fn main() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("Failed to read");
```

## **4. Error Handling (`std::result` & `std::option`)**
Rust provides two powerful enums for handling errors: `Result<T, E>` and `Option<T>`.

### **Using `Result` for Error Handling**
```tsx
<CodeBlock language="rust">
{`fn divide(x: i32, y: i32) -> Result<i32, String> {
    if y == 0 {
        Err("Cannot divide by zero".to_string())
    } else {
        Ok(x / y)
    }
}

fn main() {
    match divide(10, 2) {
        Ok(result) => println!("Result: {}", result),
        Err(e) => println!("Error: {}", e),
    }
}`}
</CodeBlock>
```

### **Using `Option` for Optional Values**
```tsx
<CodeBlock language="rust">
{`fn find_value(vec: &Vec<i32>, index: usize) -> Option<i32> {
    vec.get(index).cloned()
}`}
</CodeBlock>
```

## **5. Collections (`std::collections`)**
Rust provides several collection types, such as:

- **`Vec<T>`**: A growable array.
- **`HashMap<K, V>`**: A key-value storage.
- **`HashSet<T>`**: A unique collection of values.
- **`BTreeMap<K, V>`**: A sorted key-value store.

### **Using a `Vec<T>`**
```tsx
<CodeBlock language="rust">
{`let mut numbers = vec![1, 2, 3];
numbers.push(4);
println!("{:?}", numbers);`}
</CodeBlock>
```

### **Using a `HashMap<K, V>`**
```tsx
<CodeBlock language="rust">
{`use std::collections::HashMap;

let mut scores = HashMap::new();
scores.insert("Alice", 50);
scores.insert("Bob", 70);
println!("Alice's score: {}", scores["Alice"]);`}
</CodeBlock>
```
